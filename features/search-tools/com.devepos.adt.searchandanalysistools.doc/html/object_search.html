ABAP Object Search (DevEpos)

The ABAP Object Search (DevEpos) is similar to the ABAP Quick search (i.e. STRG + SHIFT + A ).

Like the ABAP Object Search that is shipped with the ADT it is integrated in the eclipse Search dialog, which can be called via STRG + H or the eclipse menu Search âž  Search...

In comparison to the ADT ABAP Object Search, the DevEpos ABAP Object Search is much more restrictive in terms of available Object Types you can search.
But this restriction also comes with the advantage of many filter capabilities that are specifically created for each search type.

Currently the following Object Type Searches are possible:

  • Database Table
  • View
  • CDS View
  • Class/Interface
  • Method
  • Message
