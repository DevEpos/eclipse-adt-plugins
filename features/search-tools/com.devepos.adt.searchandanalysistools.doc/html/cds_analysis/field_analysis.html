Field Analysis

The Field Analysis provides an Analysis for a Field of a CDS View (or Database Table/View).
The user interface of the analysis is split into two controls:

  • a viewer which shows the available fields of the chosen entity
  • a viewer that shows the analysis result of the currently selected field

To switch the analysis of a field from Top-Down to Bottom-up you can use the radio button control in the toolbar of the detail viewer.

For obvious reasons a Top-Down analysis is only available for fields of CDS views

View Options

The layout of the two viewers can be switched via the view menu to either automatic, horizonal or vertical.

The Where-Used mode only shows the usage in fields. If a field is only used inside a Join Condition or in the Where-Clause the CDS view will not be listed as a usage

Additional options

Per default only field usages in non-calculated fields are shown (i.e. fields whose origin can be clearly associated to a concrete source entity). It is however possible to enable this via the option Search in calculated Fields ().
