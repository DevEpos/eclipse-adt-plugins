Field Analysis

The Field Analysis provides an Analysis for a Field of a CDS View (or Database Table/View). You can either display the field origin for a given field or show where a field is used

The Where-Used mode only shows the usage in fields. If a field is only used inside a Join Condition or in the Where-Clause the CDS view will not be listed as a usage
