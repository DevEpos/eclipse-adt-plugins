Top Down Analysis

The Top-Down Analysis allows you to navigate the whole Dependency tree of CDS View down to the base tables.
Expansion of aCDS View node in the tree will result in the loading of that CDS Views used entities.

Custom Options

The output of the analysis can be controlled via some option, which are available via the view menu

  • Show Descriptions - To show/hide the description of an entity in the tree
  • Show Alias Names - To show/hide the Alias Name of an entity (if there is one)
  • Show Associations - To enable an extra node for the associations of a CDS View

You also have the possibility to jump to the Colors and Fonts preferences to change the color for the Alias Name
