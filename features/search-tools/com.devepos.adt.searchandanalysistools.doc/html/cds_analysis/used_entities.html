Used Entities Analysis

This feature is only available starting from NW 7.50

The Used Entities Analysis provides an aggregated view of all used database entities of a given CDS View. Additionally it shows metrics for each used entity like Joins, Number of Tables or Number of function calls.

Available metrics

Currently the following metrics are visible as columns:

  • Occurrences
  • CDS View count
  • Table function count
  • View count
  • Table count
  • Join count
  • SET operation count (e.g. union, except and intersect)
  • GROUP BY count
  • Function call count
  • CASE expresssion count
  • CAST expression count

Text Filter

The view also provides the option to filter the results with a Text Filter, which can be activated via the View Menu or the Shorcut Strg+F
