Used Entities Analysis

The Used Entities Analysis is an aggregated view on top of the Data provided by the Dependency Analyzer of ADT.
It quickly shows you all entities used in a given CDS View, their occurrences, used entities, joins and unions.

Text Filter

The view also provides the option to filter the results with a Text Filter, which can be activated via the View Menu or the Shorcut Strg+F
