Samples for CDS View Search

Searches for all CDS views that select from tables mara or marc, also the maximum result count is raised to 500

Query Field / Option Value
Object Filter frommara,marc
Maximum Results 500

Searches for all CDS views that have an association to entities that start with "i_prod" and have a released api state

Query Field / Option Value
Object Filter associ_prod* apireleased

Searches for all CDS views that have parameters and are of type "hierarchy" and whose name starts with "i_"

Query Field / Option Value
Object Name i_*
Object Filter paramstrue type hierarchy

Searches for all CDS Views whose Description contains the string "Mater"

Query Field / Option Value
Object Filter desc*Mater*

Searches for all CDS Views that have an annotation like "analytics*query"

Query Field / Option Value
Object Filter annoanalytics*query

Searches for all CDS Views that have an annotation @VDM.private: true or an Annotation like *query*: true

Query Field / Option Value
Object Filter annovdm.private*=true,*query*=true

Samples for Database Table Search

Searches for all database tables that have a field named "MATNR", whose description contains "Cond", and are located in a development package starting with "TEST"

Query Field / Option Value
Object Filter fieldmatnr desc*Cond* packagetest*

Searches for all database tables where the responsible person is the currently logged on user and which start with the name "CS"

Query Field / Option Value
Object Name cs*
Object Filter ownerme

Samples for Class/Interface Search

Searches for all Shared Memory Area Classes and Exception Classes, also the maximum result count is raised to 500

Query Field / Option Value
Object Filter catexception,area_class
Maximum Results 500

Searches for all Classes/Interfaces that have attributes that starts with c_ and whose initial value contains constant

Query Field / Option Value
Object Filter attrc_*=*constant*

Searches for all Classes/Interfaces that have the methods "get_field" and "set_field"

Query Field / Option Value
Object Filter methget_field,set_field
Combine Options of Filters with AND true

Samples for Method Search

Search for all "post" methods in classes that inherit from "cl_adt_rest_resource" or one of its sub classes

Query Field / Option Value
Object Filter supercl_adt_rest_resource
Method Name post<
Mode for Hierarchy Filter Resolve Type Hierarchy for 'super' Filter

Search for all methods whose implementation was changed "yesterday" from the currently logged on user

Query Field / Option Value
Method Filter changedyesterday changedbyme
Set Focus of Admin-Data Filters to Method Include true

Search for all methods that start with "set_", that are "final redefined" and have visibility "protected"

Query Field / Option Value
Method Name set_
Method Filter flagfinal statusredefined visibilityprotected
