Create new Search Favorite

To create a new Search Favorite you have to navigate to the Search View and select the appropriate Query from the History or execute a new Search Query via the Eclipse Search Dialog (Strg+H)
If the chosen result supports the Search Favorites you should see a yellow star Icon in the toolbar of the Search View.

Opening the dropdown menu will present you with a list of available Search Favorites

After that you will see an additional action Create Favorite from current Query (1) which will open the following dialog

Before you can save the favorite you have to provide a description for favorite. Additionally you can make the new favorite Project Independent.

Project Independent Favorites

If activated for a favorite, the search will not be immediately executed. Instead it will be opened in the Search Dialog, so that you are still able to manually change the target ABAP project.
