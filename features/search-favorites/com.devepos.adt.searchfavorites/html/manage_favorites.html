Manage Favorites

The Favorites Manager can be accessed from the main Eclipse toolbar or from the Search View.

Upon pressing the icon, a modal dialog will appear.

Options

In this dialog you can change the order of your favorites by simple drag-n-drop or by using the buttons Move Up and Move Down. You can also delete favorites by marking one or several entries and pressing the Remove Button.

To execute a Search from a Favorite you can either do a douple mouse click on an entry or you can select one and press the Open Button in bottom of the dialog.

You have also the option to control the visibility of the favorites, i.e. the favorites that should be shown in the drop-down menu that is accessible from the main toolbar or the Search Result view.

Available Actions in the Drop-Down Menu

Instead of pressing the command directly you can also open the dropdown menu of the command where you have the following options

  • Select a saved search favorite
  • Import Favorites from an XML file
    If favorites with the same description, destination Id (aka ABAP project) and search type already exist in the workspace and in the import file, they will be skipped.
  • Export the current list of Favorites to an XML file
