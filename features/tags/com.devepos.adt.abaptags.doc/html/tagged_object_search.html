Tagged Object Search

The ABAP Tagged Object search allows the searching for tagged Repository objects. It is integrated in the Eclipse Search dialog, which can be called via STRG + H or the eclipse menu Search âž  Search...

The User Interface

The Search Dialog Page is structured into the following controls

  1. A filterable tree with all available tags in the selected project
  2. A Checkbox to only consider result that match all selected tags
  3. A text input control to enter the target ABAP Project. The control supports content assist
If you want to quickly select all the child Tags of a given Sub Tree you can do this via the context menu

The Result View

Once the query is finished the results are displayed in the Search View.

Custom Toolbar Actions

In addition to the default toolbar actions you can reopen the query behind the result in the Search Dialog.
There is also the option to save the current query as a favorite (more info on Favorites)
