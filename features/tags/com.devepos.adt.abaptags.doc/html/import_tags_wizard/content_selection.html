Content selection

Use this page to select the ABAP Tags and their associated tagged objects that you wish to import. This allows you to efficiently manage and organize your imported content.

The page consists of two panels:

  • Tree control for the available tags
    • Displays a hierarchical list of tags that have been extracted from the specified source project or file
  • Table with the assigned objects to a given tag
    • Shows the specific ABAP objects associated with the selected tags.
    • Columns:
      • Object: Technical name of the object
      • Type: The type of object (e.g., Access Control).
      • Parent Tag: The tag of the parent object
      • Parent Object: The name of the parent object, to which this object is assigned
      • Parent Type: The type of the parent object, to which this object is assigned
    • Objects that are checked will be imported.
    • Marking an object for import will automatically mark the corresponding tag and the tag's parent tags for import
    • If an object has values in the Parent Type and Parent Object columns, marking it for import will also automatically select its tagged parent for import. Conversely, if a tagged object is unmarked for import, all of its dependent child objects will also be unmarked.

Options

  • Include information about shared tags
    If this option is selected, the shared users information will also be imported
    This option will not affect any shared tags that are already in the target systems

Keyboard shortcuts

You can use the following keyboard shortcuts to quickly expand or collapse all tags in the content selection page:

Shortcut Function
Ctrl + → Expand All
Ctrl + ← Collapse All
