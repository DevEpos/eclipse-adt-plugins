Tagged Components

Tagged components (e.g. local classes/interfaces of a global class) are displayed in a special way.
Instead of the name of the repository object you will see the local class name after the icon. The owning parent object - the global class - will be displayed in a customizable color afterwards.

Adjust Color of Parent Object

To customize the color of the parent object go to General > Appearance > Colors and Fonts and navigate to the category ABAP Tags.
